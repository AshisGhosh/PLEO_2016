Pleo Firmware

This folder contains the last version of each major Pleo firmware release. There
have been other interim versions released to end-users, but it is recommended that
only these versions are supported.

Versions include:

2.0.0 initial release for Pleo RB


The files are as follows:

pleo.img: firmware image for main Atmel processor in Pleo. This is used by
    the monitor 'upgrade' command, or the SD Card updater.
    
pleo[pm].urf: the default Pleo personality. Upgraded through the monitor 'resource
    upgrade' command, or the SD Card updater.
